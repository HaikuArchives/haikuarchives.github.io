    How to Upload a Repo

    1. Import source code of repository.
      • If importing from a SCM repo:
        • For SVN: git svn clone to clone the repo.
        • For HG: hg clone to clone the repo, hg-git to push it to GitHub.
      • If importing from a zip/tar/etc.:
        1. Extract archive. Delete ALL compiled files.
        2. Move all source files into a "src" directory if they aren't already. Leave README/TODO/LICENSE/etc. in top directory.
        3. Move HTML docs, if any, to their own subfolder of the parent directory.
    2. Create the GitHub repo.
      • Naming convention: Remove spaces & separators; change captialization. (e.g. "Haiku on a Stick" → "HaikuOnAStick", "Open Sum-IT" → "OpenSumIt")
    3. Push code to GitHub.
    4. Create the description for the repo (& add url if there is one). The summary probably can be extracted from the first sentence of the README. (Look at some repos on HaikuArchives for examples.)
    5. Disable the Wiki. (We can re-enable it later if we ever want it, and we aren't going to use it now.)
